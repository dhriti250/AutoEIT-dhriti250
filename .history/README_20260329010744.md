# AutoEIT GSoC 2026 - Evaluation Test II

## Overview
This project implements an automated scoring system for Elicited Imitation Task (EIT) responses.

## Approach
- Text normalization (lowercase, punctuation removal)
- Similarity-based comparison using SequenceMatcher
- Rule-based scoring (1–5 scale)

## How to Run
1. Place dataset in `data/` folder
2. Run notebook `notebooks/autoeit_test.ipynb`
3. Output will be saved in `output/`

## Output
- Excel file with sentence-level scores for each participant
